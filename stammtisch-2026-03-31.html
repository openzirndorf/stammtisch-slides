Anwendung

FE
Frontend — React 19 + Vite + Tailwind
TypeScript · React Router v7 · Lucide Icons · statisch gebaut, via Scaleway serviert
BE
Backend — Python FastAPI
SQLAlchemy · asyncpg · Pydantic v2 · REST-API · Geocoding
DB
Datenbank — Scaleway Serverless SQL
Postgres-kompatibel · serverless · kein Datenbankserver zu verwalten · nur Kosten bei Nutzung
